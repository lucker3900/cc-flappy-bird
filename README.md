# CC Flappy Bird

一个使用 Pygame 开发的 Flappy Bird 游戏克隆版。

## 游戏特性

- 经典的 Flappy Bird 玩法
- 平滑的小鸟动画
- 背景音乐和音效
- 计分系统
- 暂停功能
- 游戏结束和重启机制

## 安装说明

1. 克隆仓库：
```bash
git clone https://github.com/你的用户名/cc-flappy-bird.git
cd cc-flappy-bird
```

2. 安装依赖：
```bash
pip install -r requirements.txt
```

3. 运行游戏：
```bash
python cc_flappy-bird.py
```

## 游戏控制

- 上箭头键/鼠标左键：让小鸟跳跃
- 空格键：暂停游戏
- 游戏结束后点击重启按钮或按空格键重新开始

## 文件结构

```
cc_flappy-bird/
```

## 依赖项

- Python 3.x
